# 코딩 없이 마우스 클릭만으로 일이 자동으로 돌아가게 만들기

## 미션 소개

매일 낭비되는 단순 반복 업무의 시간을 절약하기 위해 코딩 없이도 워크플로우를 구축할 수 있는 노코드 자동화 도구가 등장했습니다. 이번 미션에서는 **동일한 워크플로우를 서로 다른 2개 이상의 자동화 도구로 구현**하고 비교 분석한 뒤 **자유 주제로 나만의 자동화 파이프라인을 설계·구현**합니다. 단순히 기능을 구현하는 것을 넘어 Trigger(시작 이벤트)와 Action(처리 동작)의 작동 원리를 이해하고 각 도구의 장단점을 파악함으로써, 향후 AI 에이전트 연동이나 코드 기반 자동화로 실력을 확장해 나갈 수 있는 탄탄한 실무 역량을 기르는 것이 이번 과제의 핵심 목표입니다.

## 최종 결과물

### 1. [[프로젝트 1] 자동화 도구 비교 분석 보고서](proj1/01_tool_comparison.md)
- 동일한 자동화 워크플로우를 3개의 노코드 도구(Make, Zapier, n8n)로 직접 구현하고 비교 분석한 보고서입니다.
- 기획 의도(자동화 업무 정의), 도구 선정 이유, 워크플로우 흐름과 전체 아키텍처를 문서화합니다.

### 2. [[프로젝트 2] 뉴스 요약 자동화 설계 및 구현](proj2/02_news_summary.md)
- n8n Cloud를 활용하여 관심 분야(AI)의 뉴스를 수집해 생성형 AI로 요약한 뒤, 데이터베이스에 자동 적재하고 알림을 보내는 '나만의 맞춤형 자동화 파이프라인'입니다.
- 워크플로우 파일과 더불어, 정상 작동 및 예외 처리(대체 경로 작동) 과정을 증명하는 스크린샷을 포함합니다.

## 활용 방법

1. 전체 자동화 구조 및 기획 요약을 파악하려면 먼저 이 README를 읽습니다.
2. 노코드 자동화 도구에 대한 이해와 분석은 `01_tool_comparison.md`를 확인합니다.
3. 뉴스 요약 자동화 워크플로우 설계 문서는 `02_news_summary.md`를 확인합니다.
4. **[중요]** 프로젝트 2의 실제 동작 구조를 직접 확인하려면 첨부된 `n8n_workflow.json` 파일의 내용을 복사하여 n8n 빈 캔버스에 **붙여넣기(Ctrl+V)** 하거나, **Import from File** 기능을 활용해 확인합니다.
